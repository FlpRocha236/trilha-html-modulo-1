História dos computadores

Computar: A palavra computador vem do verbo computar, que significa calcular, uma das primeiras máquinas de computar foi o ábaco, criado na china no seculo V antes de Cristo. No século XIX, o matemático inglês Charles Babbage criou a primeira máquina analítica, que a grosso modo, foi comparada ao primeiro computador. Através dessa invenção foi considerado por muitos como "Pai da Informática", Evolução dos computadores, Primeira geração (1951 - 1959) Os computadores de primeira geração funcionavam por meio de circuitos e válvulas eletrônicas. Possuíam o uso restrito, além de serem imensos e consumirem muita energia. Um exemplo é o ENIAC (Eletronic Numerical Integrator and Computer) que consumia cerca de 200 quilowatts e possuía 19.000 válvulas.

O que são Clients?

Clients: é um termo empregado a entidade que solicita ou consome serviços servidos de uma entidade servidora(Server) em geral através do uso de uma rede de computadores nume arquitetura cliente-servidor. Como funciona: , o cliente e solicita O cliente solicita informações ou serviços ao servidor. O servidor fornece as informações ou serviços solicitados pelo cliente. O cliente processa os dados recebidos do servidor e recebe os resultados, O modelo cliente-servidor é fundamental para compreender como as trocas de dados ocorrem em uma rede.

O que são Servers?

Servidor: é uma entidade ou computador que fornece dados ao cliente ou usuário em uma rede de computadores, Como funciona, Processam solicitações e fornecem dados a outros computadores, Gerenciam o acesso a recursos de rede, Armazenam dados importantes, Executam aplicações críticas, Podem executar cálculos para um cliente

Como é uma estrutura básica de um HTML?

HTML: Uma estrutura básica de HTML é formada basicamente pelas tags, DOCTYPE,HTML,HEAD e BODY, sendo o DOCTYPE um documento de instrução para os navegadores e outros programas saberem executar e processar o site da melhor maneira possivel, Lembre-se: o doctype é OBRIGATÓRIO e deve ser sempre a PRIMEIRA LINHA do seu documento. O HEAD, Contém informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Na prática, todo o conteúdo do cabeçalho fica delimitado entre a abertura e fechamento. BODY, Trata-se do documento em si, ou seja, a informação legível para o usuário/leitor do documento. É todo e qualquer texto que se deseja apresentar, assim como toda e qualquer forma de mídia de saída (imagens, sons, miniaplicativos embutidos, conteúdo multimídia, etc). Além disso, toda a apresentação de entrada de dados (formulários), também se aplica nesta seção do documento.

Entendendo a diferença entre listas ordenadas e não ordenadas

Resumo: Listas Ordenadas são classificadas por numéros ou representação ordenada determinada pelo usúario, Lista não ordenadas, são o contrario disso, criando uma lista definição, somente seguindo uma sequencia, de pontos ou o que for utilizado pelo usúario.
